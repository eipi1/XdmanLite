Downloading Videos With XDM

YouTube videos can be easily downloaded with
XDM YouTube Downloader.
Goto XDM Menu-> File ->YouTube Downloader.


Downloading videos from Metacafe,Google Video,Daily Motion or any other website:

For Firefox/SeaMonkey users

  • Integrate XDM with Firefox.
    Goto XDM Menu->Browser Integration and drag the icon into Firefox's Window.This will install the XDM extension into Firefox/SeaMonkey.

  • Using Firefox/SeaMonkey visit the website (ie. Youtube,Metacafe etc.) and play the video in your Browser.

  • After the video is started playing Right click inside Firefox's Window and select 'Download FLV with XDM' from the popup menu.

  • The download link for the FLV video will be displayed in XDM Media Grabber.

  • Click on the download button.

NOTE:

  • Make sure XDM is running while playing the video in Firefox/SeaMonkey.

For Chrome, Internet Explorer, Opera, Safari or Other Browsers.

  • Configure Network for Advanced Browser Integration.
    Goto XDM Menu->Browser Integration-> Monitor All Browsers.
    If you are using Windows click on the Auto Configure button.
    If you are using Linux, Mac etc click on the Manual configuration button and follow the steps or refer to Browser_Integration.txt

  • After the first step is done close the browser and start it again to Make sure XDM starts running before the browser starts.
    To make this easy XDM can be added to startup.

  • Using Browser visit the website (ie. Youtube, Metacafe etc.) and play the video in the Browser.

  • Goto XDM Menu->File->Grabber to download the video.
