Advanced Browser Integration

XDM can be intgrated with any browser to capture downloads. This feature also enables XDM to grab videos from websites like Youtube, Metacafe etc.This section describes how to enable this feature.

Activating Advanced Browser Integration on Linux, Mac or other Opearing Systems

  • Firefox/SeaMonkey
    Goto XDM Menu->Tools->Browser Integration->
    'Monitor Firefox' and Install the extension
  • Google Chrome
    Goto Chrome's Settings page ->Show Advanced Settings ->
    Network->Change proxy settings.
    Select Proxy Auto Configuration or similar option and provide
    Auto Proxy URL as 'http://localhost:9614/proxy.pac'
  • Opera
    Goto Preferences->Advanced->Network->Proxy Servers->Check
    automatic proxy configuration and provide the address
    'http://localhost:9614/proxy.pac'
  • Konqueror
    Goto Settings->Configure->Proxies and provide auto proxy url as
    'http://localhost:9614/proxy.pac'
  • Apple Safari
    Goto Preferences->Advanced->Change settings
    Check 'Automatic Proxy Configuration' and enter the address
    'http://localhost:9614/proxy.pac'
  • Other Browsers
    Goto Configuration/Preferences->Network->Proxy settings
    And provide the auto proxy address 'http://localhost:9614/proxy.pac'
NOTE: Always start XDM before starting your browser.
You may add XDM on System start-up.
Check 'Run XDM on start-up' from XDM's browser integration dialog.

Activating Advanced Browser Integration on Windows
XDM will autometically enable this feature while running on Windows. NO manual steps are required. However if autometic configuration failed for some reaseon, you still can use this feature by manually configuring the network. You can turn this feature on/off anytime using: XDM Menu->Tools->Browser Integration-> Monitor All Browsers and click on the Auto Configure Button.
NOTE: XDM will be able to takeover downloads from browsers while XDM is running.

To configure manually, perform the following steps:
1. Goto Control Panel->Internet Option->Connections. If you are using Dialup or VPN then select the connection from the list and click on the Settings button. If you are using LAN then click on LAN settings.
2. Select use Auto configuration Script checkbox and type http://localhost:9614/proxy.pac in the textbox.
3. Save the settings.
NOTE:
1. Manual configuration will work properly only if XDM runs on System start-up. You can select this option in XDM Menu-> Browser Integration Dialog.
2. To capture download from Opera, start XDM before starting Opera.
3. To capture download from Firefox, install the firefox extension from Browser Integration Dialog.
